Enter a YouTube URL and click "Convert" to start.

How it works

  1. Enter a valid YouTube video URL in the input field
  2. Click "Convert" to send the URL to our processing server
  3. Wait while the server:
    • Downloads the audio from YouTube
    • Converts the audio to text using speech recognition
  4. View and download the transcribed text

Note: This frontend is hosted on GitHub Pages. The actual processing happens on a separate backend server.
